AUコンバータ ver. 1.07
wave形式の音声データをau形式に変換

　Windowsにおける音声データの標準形式であるwave形式のファイルを、Javaで利用可能なau形式に変換します。プログラムに組み込みたいwave形式のファイルを、このツールでau形式に変換できます。
　このツールはWindows(Windows95以降)用のプログラムです。他のOSをご利用の場合は、それぞれのOS上で動作する同様のツールを、インターネット上から見つけて下さい。

インストール
　lzh形式なので、解凍後、適当なディレクトリに置いて下さい。

使用方法
　解凍して得られるaucon.exeに、wave形式のファイルをドラッグ&ドロップすることで、au形式への変換を行います。
　詳細については、aucon添付の文書を読んで下さい。
　添付の文書に目を通し、利用条件などを確認して下さい。
　音声データは著作物です。利用に関しては、十分に注意を払いましょう。

作者：参納 大樹
参照URL：http://hp.vector.co.jp/authors/VA015468/

eo ver. 1.5.2
.lzh、.zip形式などの書庫を解凍

　複数のファイルをまとめて配布するために、複数のファイルを1つのファイルにしたものを書庫(アーカイブ)ファイルといいます。
　eoは、一般的な書庫ファイルフォーマットであるlzh形式、zip形式をはじめ、様々な形式の書庫を簡単に解凍(元の複数のファイルに戻す)するツールです。
　本CD-ROM収録物には、lzh形式やzip形式の書庫が含まれていますので、もしあなたがこれらの解凍を行うツールを持っていないのであれば、このeoを使って下さい。
　このツールはWindows(Windows95以降)用のプログラムです。他のOSをご利用の場合は、それぞれのOS上で動作する同様のツールを、インターネット上から見つけて下さい。

インストール
　実行すればインストールできます。

使用方法
　標準的な使用方法では、ディスクトップに作られるeoのショートカット(アイコン)に書庫ファイルをドラッグ&ドロップすることで、書庫の解凍を行います。
　詳細については、eo添付の文書やヘルプを読んで下さい。
　添付の文書に目を通し、利用条件などを確認して下さい。

作者：手賀 俊行
参照URL：http://member.nifty.ne.jp/beambitious/

IrfanView32 日本語版 ver. 3.85
 gif形式の画像ファイルを生成

　様々な形式の画像ファイルに対応した、画像ファイルビューアです。画像形式を変換して保存する機能も備えており、フリーウェアには珍しく、gif形式の画像を保存することができます。透過gifにも対応しています。
　オリジナル版は Irfan Skiljan 氏がオーストリアにおいて開発し、その日本語版モジュールを楠本拓矢氏が開発し、日本語版パッケージが配布されています。
　Javaのゲームプログラム内で使用するために用意した画像を、gif形式に変換したい場合に使うとよいでしょう。　このツールはWindows(Windows95以降)用のプログラムです。他のOSをご利用の場合は、それぞれのOS上で動作する同様のツールを、インターネット上から見つけて下さい。商用ソフトウェアやシェアウェアの試用版でも、簡単な用途には足るかも知れません。

インストール
　zip形式なので、解凍後、適当なディレクトリに置いて下さい。

使用方法
　解凍して出来るディレクトリ内の i_view32.exe を起動します。起動したら、ウィンドウに、gifに変換したい画像ファイルをドラッグ&ドロップします。ウィンドウに画像が表示されたら、[ファイル]-[名前を付けて保存]を選択し、保存する画像形式をgifに設定し、保存します。この時、SaveTransparentColor の設定を行えば、透過指定を行うことができまず。詳細については、配布元のwebページや、検索サイトで見つかる使い方を解説したwebページを参考にして下さい。
　添付の文書に目を通し、利用条件などを確認して下さい。

日本語版作者：楠本 拓矢
参照URL：http://www8.plala.or.jp/kusutaku/

オリジナル版作者：Irfan Skiljan
参照URL：http://www.irfanview.com

JavaSDK 
Windows用（Java2SDK ver. 1.4.2_03）
 Linux用（Java2SDK ver. 1.4.2_03）
 Linux用RPM形式（Java2SDK ver. 1.4.2_03）

　Javaのプログラムをコンパイルしたりするのに必要なSDKです。これがないと始まりません。

インストール
　Windows版、Linux版、何れも実行すればインストールが行えます。Linux版の場合、ファイルに実行属性を付けるか、シェルスクリプトファイルとして実行して下さい。
・WindowsへのSDKのインストール
・MacOS9へのSDKのインストール
※MacOSXでは、OSに付属するJavaの開発環境が利用できます。但し、開発環境のアップデートが行われているので、Appleのwebサイトから最新版を手に入れるようにした方がよいでしょう。開発環境を入手するためには、ADC（AppleDeveloperConnection）会員に登録する必要があります。登録は無料で、Appleのwebサイト上で行えます。
　開発に必要なツール類がインストールできたら、マニュアル類を入手し、インストールする必要があります。本書執筆時点で、最新版の日本語ドキュメントは、Java1.4.0 のものが、Sun Microsystems のwebサイトからダウンロードできます。zip形式なので解凍し、適当なディレクトリに置き、index.html をwebブラウザで開いて下さい。
最新版の日本語ドキュメント：http://java.sun.com/j2se/1.4.2/ja/download.html

収録されているSDKの作者：Sun Microsystems
参照URL：http://sdc.sun.co.jp/java/

TeraPad ver. 0.85
Windows用テキストエディタ

　プログラムの作成や、文書の作成など、Windows上でテキストファイルを作成、編集する作業のためには、使いやすい高機能なテキストエディタがあると便利です。
　Windows付属のテキストエディタ「メモ帳(notepad)」は、使いやすい高機能なテキストエディタであるとは言いにくいので、このTeraPadのようなテキストエディタを入手して使用したほうがよいでしょう。
　このツールはWindows(Windows95以降)用のプログラムです。他のOSをご利用の場合は、それぞれのOS上で、満足できるテキストエディタを、インターネット上から見つけて下さい。

インストール
　実行すればインストールできます。

使用方法
　terapad.exeを起動したら、[ヘルプ]-[Readme.txt]を見て下さい。更に、html形式のオンラインヘルプもあるので、そちらも役立ちます。
　添付の文書に目を通し、利用条件などを確認して下さい。

参照URL：http://www5f.biglobe.ne.jp/~t-susumu/
